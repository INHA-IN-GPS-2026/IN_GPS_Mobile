# IN_GPS_Mobile (Android)

IN_GPS Mobile입니다. (only Android OS)
써미스터의 온도를 실제 사용자가 확인할 수 있는 화면을 제공합니다.
## Project Structure
 - app/kotlin+java
   - api/ : 서버 데이터 파싱
   - model/ : DB TABLE
   - screen/ : 화면관련 코드 
   - utils/ : 추가중
 - res/ :xml 파일들
   - layout/ : 화면 로드
   - 
## Features
- web API Data Parsing
- Graphing

## Requirements
- Android OS
- Java & Kotlin
- Android Studio
- xml

## Build
- download .apk file or run to Android Studio
